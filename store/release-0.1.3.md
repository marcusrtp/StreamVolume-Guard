# StreamVolume Guard 0.1.3

Version publique de test centrée sur la stabilité audio de la page de test, l'égalisation faible / fort / très fort et la cohérence des builds.

## Points clés

- Page de test audio plus propre : un seul WAV continu avec segments faible, fort et très fort.
- Moins de grésillements lors du passage de `Son très fort` à `Son fort`, même sans extension activée.
- Smoke test navigateur aligné avec les vrais niveaux audio encodés.
- Builds Chromium, Firefox, Firefox Android et source Safari régénérés.
- Politique de confidentialité publique ajoutée dans `docs/privacy-policy.md`.
- Plan de test plateformes réelles ajouté pour YouTube, Twitch, TikTok, Kick, Spotify web et Deezer web.
- Script `tools/package-release.js` ajouté pour générer les zips de release de façon reproductible.

## Fichiers recommandés

- Projet complet : `streamvolume-guard-project-0.1.3.zip`
- Chrome, Brave, Edge desktop : `streamvolume-guard-chromium-0.1.3.zip`
- Firefox desktop : `streamvolume-guard-firefox-0.1.3.zip`
- Firefox Android : `streamvolume-guard-firefox-android-0.1.3.zip`
- Safari macOS / iOS / iPadOS : `streamvolume-guard-safari-source-0.1.3.zip`

## Confidentialité

StreamVolume Guard reste open source, sans tracker, sans télémétrie automatique et sans collecte d'audio, d'historique ou de données personnelles.
